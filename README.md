JavaScript Calculator

This project is a simple JavaScript calculator that performs basic arithmetic operations and keeps a history of all calculations. It includes an HTML interface and a JavaScript file containing the calculator logic.

Features

Addition, subtraction, multiplication, and division

Divide-by-zero error handling

History tracking with timestamps

Clean and simple user interface

Dynamic history display on the page

Files

index.html — User interface

calculator.js — Calculator functions + history storage

README.md — Project documentation

How to Use

Open index.html in your browser.

Enter two numbers.

Click an operation button.

View the result and calculation history below.
